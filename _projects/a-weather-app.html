HOW WE BUILT IT

Overview

For the first bit of 2018, we focus on learning React. So we took a course on Lynda and Codecademy and felt comfortable to try to make something afterwards. We chose a pretty straightforward beginner project, a weather app. We wanted to use real data, received from an API, and display the weather. We also let the user search for a location and displayed images from Unplash. This basic project showed us the foundation of getting data, displaying it, and interacting with it, so that we have a foundation for next time.
